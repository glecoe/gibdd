# Приложение сотрудника — ГИБДД

Android-приложение для инспекторов, администраторов и начальников.
Логин по телефону/паролю, патрулирование, обработка инцидентов от очевидцев,
управление пользователями и ролями.

## Функции по ролям

**Инспектор:**
- Вход по телефону/паролю
- Кнопка «Начать / Закончить патруль» (на патруле приходят инциденты)
- Лента инцидентов: принять в работу, закрыть
- Просмотр фото/видео и геолокации от очевидца

**Администратор** (всё что инспектор +):
- Экран управления пользователями
- Назначение ролей: инспектор / очевидец
- Создание инспекторов
- Тумблер уведомлений от очевидцев (только для себя)

**Начальник** (всё что админ +):
- Назначение/снятие администраторов и начальников
- Создание сотрудников любой роли
- Отключение (деактивация) пользователей

## Сборка

1. Открой папку `officer_app` в **Android Studio**.
2. Дождись синхронизации Gradle.
3. Run → Run 'app'.

> Приложение собирается и **без** Firebase. В этом случае пуши не приходят,
> но уведомления подхватываются опросом сервера каждые 15 секунд, пока
> инспектор на патруле. Для мгновенных пушей настрой Firebase (ниже).

## Первый вход

На сервере при первом запуске создаётся начальник:
- телефон: `+70000000000`
- пароль: `admin123`

Войди под ним, затем через экран «Управление» создавай инспекторов и админов.

## Настройка адреса сервера

По умолчанию `http://10.0.2.2:8000/` (хост-машина из эмулятора).
Для реального телефона: Настройки (шестерёнка на экране входа) →
введи `http://<IP компьютера>:8000` → Сохранить. Телефон и компьютер
в одной Wi-Fi сети, сервер запущен с `--host 0.0.0.0`.

## Карты (OpenStreetMap)

Карта показывается на отдельном экране при тапе на координаты инцидента.
Используется библиотека **osmdroid** с тайлами **OpenStreetMap** — никаких
API-ключей не нужно, никакой регистрации, никакого биллинга. Просто работает.

Тайлы качаются с серверов OSM при первом отображении и кэшируются на устройстве.
Требуется только доступ к интернету (уже в манифесте).

## Настройка Firebase (для пушей) — опционально

1. Зайди на https://console.firebase.google.com, создай проект.
2. Добавь Android-приложение с package name **`com.gibdd.officer`**.
3. Скачай файл **`google-services.json`** и положи его в папку `officer_app/app/`.
   (Как только файл появится — Gradle сам подключит Firebase-плагин.)
4. На сервере:
   - В настройках проекта Firebase → Service accounts → Generate new private key,
     скачай JSON-ключ.
   - Установи на сервере: `pip install firebase-admin`
   - Задай переменную окружения перед запуском сервера:
     `GOOGLE_APPLICATION_CREDENTIALS=/путь/к/serviceAccount.json`
5. Пересобери приложение и перезапусти сервер. Теперь при создании инцидента
   инспекторам на патруле летят пуши.

## Стек

- Kotlin 2.0 + Jetpack Compose + Material3
- Retrofit 2 + OkHttp
- DataStore Preferences
- Coil (фото в ленте)
- Media3 ExoPlayer (просмотр видео)
- osmdroid (карта OpenStreetMap)
- Firebase Cloud Messaging (опционально)
- Navigation Compose
